3.1 Dearly Data'd Descendants
I know, I know: I promised at the end of Lesson 1 you would get to see some crazy animations soon. Well, soon isn't yet. Soon is in the future. Now, sadly, we get to spend an entire lesson not making eyeballs bleed and potential romantic partners swoon. We are just going to take a deep dive into data (sorry to the rebels; there will be no sidetrack into lore).
Back in the first tutorial, there was a short example involving sugar, spice, and appending nodes. We are going to revisit that, but as a data-driven example:
<style type='text/css'> .thing { border: 1px solid black; margin: 5px; padding: 5px; width: 200px; } .thing.nice { background: #cfc; } .thing.notnice { background: #fcc; } </style> <div id='chart'></div> var things = [ {name: 'Sugar', isNice: true}, {name: 'Spice', isNice: true}, {name: 'Toe Fungus', isNice: false}]; d3.select('#chart').selectAll('div.thing') .data(things).enter() .append('div') .classed({ 'thing': true, 'nice': function(d) { return d.isNice; }, 'notnice': function(d) { return !d.isNice; }, }) .text(function(d) { return d.name; });
